# Location Graph (PA04) — Java

This is the original graph project code (from `PA04.zip`), implemented as an undirected weighted graph with:

- `addLocation(...)`
- `addDistance(...)` (adds edges in both directions)
- `findDistanceBreadthFirst(...)`
- `findDistanceDepthFirst(...)`
- `detectCycle()`
- `findMinimumPath(...)` (Dijkstra-style shortest path)

In this portfolio site, I ported the same behaviors into a live in-browser demo so recruiters can interact with inputs and see outputs instantly.
