# Neurodivergent Settings Recommender

## Project intent

This project focuses on an interactive settings recommender for neurodivergent-friendly web experiences. Users choose preferences (focus, visual sensitivity, reading support, navigation style, pacing), and the interface returns practical accessibility recommendations.

## UX case study highlights

- **Problem:** Many websites create cognitive overload with cluttered layouts, long text blocks, and unpredictable interactions.
- **Research:** Used cognitive accessibility guidance focused on reduced motion, plain language, chunked content, and personalization.
- **Design decisions:** Built a question-based flow that maps user preferences to clear recommendations and short explanations.
- **Testing + iteration:** Simplified wording and added example scenarios so recommendations are easier to understand quickly.
