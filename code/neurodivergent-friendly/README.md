# Neurodivergent-Friendly Website Prototype

Portfolio project link:
https://briepo.myportfolio.com/wireframe-prototypes

## Project intent

This project explores how website experiences can better support neurodivergent users by reducing cognitive overload and making navigation more predictable.

## UX case study highlights

- **Problem:** Many websites are visually noisy and hard to scan, which can create friction for users with different cognitive processing styles.
- **Research:** Reviewed accessibility-centered patterns and user comfort factors tied to readability, hierarchy, and information chunking.
- **Design decisions:** Prioritized calmer layout structure, stronger spacing rhythm, cleaner typography choices, and clearer call-to-action hierarchy.
- **Testing + iteration:** Refined content grouping and interaction flow after feedback to improve first-pass clarity and task confidence.
