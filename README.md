# Human-Centered Design Portfolio (GitHub Pages)

This repo powers a clean, creative portfolio site with interactive project demos.

## How to deploy

1. Create a GitHub repo from this folder.
2. Turn on **GitHub Pages**.
3. Set the Pages source to the **root** (this uses `index.html` at the repo root).
4. If you deployed the Streamlit Art Gallery chatbot somewhere, paste the public URL into `scripts/app.js` → `PROJECTS[1].embed.url`.

## Projects included

1. `Location Graph Explorer` (interactive in-browser port)
2. `Art Gallery AI Chatbot` (Streamlit + LangChain embed placeholder)
3. `Accessify (HCDD Prototype)` (Figma embed placeholder)
4. `Bakery Storefront (SQL + Reasoning)` (MySQL schema/queries + screenshots)
